## Sobre o Projeto

Este é um servidor back-end desenvolvido com Node.js e Express para uma loja virtual de produtos de informática. O projeto demonstra a criação de um servidor web completo com rotas, páginas HTML estáticas, arquivos CSS e imagens.

### Tecnologias Utilizadas

- **Node.js** - Ambiente de execução JavaScript
- **Express** - Framework web para Node.js
- **HTML5** - Estrutura das páginas
- **CSS3** - Estilização responsiva
- **JavaScript** - Funcionalidades interativas

#### Funcionalidades

- Servidor web com Express
- Rotas configuradas para produtos específicos
- Páginas estáticas com HTML
- Estilização CSS responsiva
- Imagens e recursos visuais
- Sistema de navegação entre páginas
- Página de erro 404 personalizada
